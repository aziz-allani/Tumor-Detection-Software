# 🧠 Tumor Detection Software using Deep Learning (CNN + Django)

This project is an AI-powered web application for brain tumor detection from MRI images.  
It uses a Convolutional Neural Network (CNN) trained on medical imaging data and is deployed using Django to provide real-time predictions through a web interface.

The system classifies MRI scans into four categories:
- Glioma
- Meningioma
- Pituitary tumor
- No tumor

## 🎯 Key Features
- 🧠 Deep learning-based tumor classification (CNN)
- 📊 Trained on 7,000+ MRI images
- ⚡ Real-time prediction through web interface
- 🖼️ Image upload and result visualization
- 🔄 Data augmentation for better generalization
- 📉 Performance evaluation using accuracy, precision, recall, F1-score

## 🧠 Model Architecture
The CNN model consists of:
- Multiple Convolutional layers (Conv2D)
- MaxPooling layers for feature reduction
- Fully connected Dense layers
- Softmax output layer for classification

---

## 📊 Performance
The model achieves strong performance on MRI classification:
- High accuracy on validation data
- Balanced precision and recall across all classes
- Evaluated using confusion matrix and F1-score
